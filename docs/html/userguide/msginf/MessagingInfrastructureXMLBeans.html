The Messaging Infrastructure and XMLBeans

The Messaging Infrastructure uses XML Beans to convert the messaging infrastructure XML properties file msginf.xml into Java objects and vice versa.

XML Beans takes the XSD schema of the messaging infrastructure XML properties file and converts it to a jar file containing classes for each element in the schema file. These classes are then used to manipulate the messaging infrastructure properties.

Installing XML Beans

Download XML Beans from the XML Beans website and follow these instructions to install them.

Creating the Messaging Infrastructure JAR file

Checkout the messaging infrastructure code from SVN into a directory. This directory will be referred to as MSGINF_HOME.

From the MSGINF_HOME/xmlbeans directory, run the following command: scomp -out msginf_xmlbeans.jar msginf.xsd

This will create the msginf_xmlbeans.jar file containing the classes needed to manipulate the messaging infrastructure XML properties file. The msginf_xmlbeans.jar file is in the classpath of the messaging infrastructure.

Each time the msginf.xsd schema is changed, the msginf_xmlbeans.jar file has to be recreated.
