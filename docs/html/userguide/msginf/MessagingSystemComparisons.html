Messaging System Comparisons

This document outlines the differences between ActiveMQ, JORAM, and JBossMQ. OpenJMS was excluded as it performed very poorly.

Observational Comparisons

The following table outlines some observations from using the messaging systems.

ActiveMQ JORAM JBossMQ
No need to register JNDI objects. Need to register JNDI objects. Need to register JNDI objects.
Little security. Security can be applied to registered JNDI objects. Security can be applied to registered JNDI objects.
Faster startup in persistent mode. Startup slower in persistent mode especially with large initial messages.
Database persistence supported. No database persistence. Custom persistence would need to be written. Database persistence supported.
Memory only persistence supported. Memory only persistence supported. No memory only persistence.
